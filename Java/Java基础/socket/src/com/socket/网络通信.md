### 网络通信三要素
#### IP地址
- 网络中设备的标识
- 不易记忆
- 可用主机名
#### 端口号
- 用于标志进程的逻辑地址，不同进程的标识
- 物理端口 网卡口
- 逻辑端口 
    - 每个网络程序都会至少有一个逻辑端口
    - 用于标识进程的逻辑地址，不同进程的标识
    - 有效端口：0~65535，其中0~1024系统使用或保留端口
#### 传输协议
- 通信的规则
- 常见协议：TCP、UDP
    - UDP：将数据源和目的封装成数据包中，不需要建立连接；每个数据包的大小控制在64K；因无连接，是不可靠的协议；不需要建立连接，速度快
    - TCP：建立连接，形成传输数据的通道；在连接中进行大数据传输；通过三次握手完成连接，是可靠协议；必须建立连接，效率会稍低
